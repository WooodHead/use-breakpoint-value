# use-breakpoint-value

`npm install --save use-breakpoint-value`

`yarn add use-breakpoint-value`


```js
import useBreakpointValue from 'use-breakpoint-value';

const responsiveValue = useBreakpointValue({
  xs: 'xs-value',
  sm: 'sm-value',
  md: 'md-value',
  lg: 'lg-value',
  xl: 'xl-value',
  xxl: 'xxl-value',
});

```
